# 🚀 Introducción a Arduino

Este documento explica lo básico de un proyecto con **Arduino**, pensado para estudiantes que quieren aprender desde cero.

---

## 🔧 Requerimientos de Hardware
Para comenzar necesitarás:
- Una placa Arduino (UNO es la más común para principiantes).
- Cable USB para conectar la placa al computador.
- Protoboard (placa de pruebas).
- Resistencias y LEDs.
- (Opcional) Sensores básicos: temperatura, luz, movimiento.
- (Opcional) Actuadores: servomotores, buzzer, motores DC.

---

## 💻 IDE Recomendado
El entorno más usado es el **Arduino IDE**, disponible en:
- [Arduino IDE oficial](https://www.arduino.cc/en/software)  
También puedes usar **PlatformIO** (integrado en VS Code) si prefieres un entorno más avanzado.

---

## 👋 Hola Mundo en Arduino
El “Hola Mundo” de Arduino es hacer parpadear un LED.  
Este código enciende y apaga el LED integrado de la placa cada segundo:

```cpp
// Hola Mundo: Blink LED
void setup() {
  pinMode(LED_BUILTIN, OUTPUT); // Configura el LED como salida
}

void loop() {
  digitalWrite(LED_BUILTIN, HIGH); // Enciende LED
  delay(1000);                     // Espera 1 segundo
  digitalWrite(LED_BUILTIN, LOW);  // Apaga LED
  delay(1000);                     // Espera 1 segundo
}
