# 📖 Cookbook Arduino

Un repositorio abierto para guardar, compartir y organizar **recetas de código Arduino**.  
La idea es que funcione como un *cookbook*: cada receta es un ejemplo práctico y reutilizable que puedes copiar, adaptar y combinar en tus propios proyectos.

---

## 🎯 Objetivo
Facilitar el acceso a fragmentos de código y proyectos básicos de Arduino, organizados de manera sencilla, para que tanto principiantes como desarrolladores experimentados puedan:
- Encontrar rápidamente soluciones a problemas comunes.
- Aprender nuevas técnicas y buenas prácticas.
- Contribuir con sus propias recetas.

---

## 📂 Estructura del repositorio
- `recipes/` → Carpeta principal con las recetas de código.
- `examples/` → Ejemplos completos de proyectos.
- `docs/` → Documentación adicional, guías y notas.
- `README.md` → Este archivo, con la explicación general.

Cada receta incluye:
- Descripción breve del propósito.
- Código Arduino listo para usar.
- Notas sobre hardware necesario (si aplica).
